# Changelog

All notable changes to Repo Doctor AI will be documented here.

## 0.1.0

- Added TypeScript Node.js CLI scaffold with Commander.
- Implemented stack detection for Node, Python, Go, Rust, Java, .NET, PHP, and Ruby.
- Added deterministic repository checks across presentation, build/test readiness, CI/CD, security, and contributor readiness.
- Added weighted scoring, summary counts, and prioritized recommended fixes.
- Added terminal, Markdown, and JSON reporters.
- Added safe template generation with stack-aware CI templates.
- Added optional `repo-doctor.config.json` support.
- Added polished README, sample report, fixtures, tests, and CI.
